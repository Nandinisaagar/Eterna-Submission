# Eterna Submission

A web application for tracking and visualizing Eterna game metrics and player progress.

## Live Demo
Check out the live application: https://eterna-submission.vercel.app/

## About
This project is a submission for Eterna, providing a dashboard interface to monitor key metrics including:

- New Pairs progress
- Final Stretch milestones
- Migration status

The application displays real-time statistics and progress indicators for RNA design puzzles and challenges.

## Features

- **Real-time Metrics Dashboard:** Track important game statistics at a glance
- **Progress Visualization:** Monitor completion of different game phases
- **Responsive Design:** Works seamlessly across desktop and mobile devices
- **Clean UI:** Modern, intuitive interface for easy navigation

## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)
- npm or yarn package manager

### Installation

Clone the repository:

```bash
git clone https://github.com/Nandinisaagar/Eterna-Submission.git
cd Eterna-Submission
```

Install dependencies:

```bash
npm install
# or
yarn install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open **http://localhost:3000** in your browser to see the application.

## Built With

- **Frontend Framework:** Modern JavaScript/TypeScript framework
- **Deployment:** Vercel
- **Styling:** CSS/Styled Components
